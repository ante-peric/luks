2. Instalacija LUKS Paketa


Na radnoj stanici koja će se koristiti vršimo instalaciju crypt paketa.


Instalacije se vrši sa naredbom:


CentOS 7:


yum -y install cryptsetup


CentOS 8:


dnf -y install cryptsetup


Važno:


LUKS formatiranje briše sav sadržaj na disku I tu trebamo biti oprezni kad koristimo LUKS. Kreiranje LUKS diskova/particija se može raditi I na VM (Virtual Machine ) kao I na stvarnoj radnoj stanici.


Aktiviramo LUKS modul


modprobe dm_crypt


Provjeravam da li je modul pokrenut


[root@server ~]# lsmod | grep dm_crypt

dm_crypt 27346 0

dm_mod 124461 17 dm_log,dm_mirror,dm_crypt
